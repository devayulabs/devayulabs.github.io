How to Enable USB Debugging and Execute ADB Commands

Step 1: Enable Developer Options

  1. Open the Settings app on your Android device.
  2. Scroll down and select About phone.
  3. Find the Build number entry.
  4. Tap the Build number entry seven times. You may be prompted to enter your device's PIN or password.
  5. You should see a message saying "You are now a developer!" or "Developer mode has been enabled".

Step 2: Enable USB Debugging

  1. Go back to the Settings menu.
  2. Scroll down and select System (or Additional settings).
  3. Tap on Developer options.
  4. Scroll down and find the USB debugging option.
  5. Toggle the USB debugging switch to turn it on.
  6. Confirm the action if prompted.

Step 3: Install ADB on Your PC

  1. Download the ADB platform-tools package for your operating system.
  2. Extract the downloaded ZIP file to a location on your PC.
  3. Open a command prompt or terminal window.
  4. Navigate to the directory where you extracted the platform-tools.

Step 4: Connect Your Android Device to Your PC

  1. Connect your Android device to your PC using a USB cable and select file transfer in USB Settings.
  2. You may see a prompt on your device asking you to allow USB debugging. Tap OK to allow it.

Step 5: Execute ADB Commands

To verify that your device is connected properly, run the following command in the command prompt or terminal:

adb devices

You should see a list of connected devices. If your device is listed, you can start executing ADB commands to grant Overlay permission.

Step 6: Grant Overlay Permission to Game Mode

To grant overlay permission to the com.devayulabs.gamemode app, execute the following ADB command:

adb shell pm grant com.devayulabs.gamemode android.permission.SYSTEM_ALERT_WINDOW

Final Step: Open the App and Enable Game Panel

Finally, open the Game Mode app on your Android device and enable the Game Panel feature.
